# 2026 下半年 13 周备考计划

本计划从 2026-07-22 开始，覆盖到广东考区下半年准考证打印结束日 2026-10-23。若开始日期较晚，优先保留“真题复盘、案例训练、论文成稿、全真模拟”，压缩纯阅读时间。

## 训练指标

每周至少记录：

- 学习小时数；
- 分模块选择题正确率；
- 案例题完成数量及丢分原因；
- 论文提纲/全文数量；
- 本周新增和关闭的薄弱点；
- 一次键盘长时间作答训练的状态。

## 错题归因标签

每道错题只能选择一个主因，避免只写“粗心”：

- `K0`：不知道知识点；
- `K1`：概念边界混淆；
- `C0`：不会计算或推导；
- `S0`：没有识别场景约束；
- `T0`：时间分配失败；
- `R0`：题干或选项阅读错误；
- `E0`：知道结论但表达不含评分关键词。

每周优先解决出现次数最多的两个标签，而不是无差别增加题量。

## 论文素材准备

建议至少准备三个可交叉复用的真实项目：

1. 高并发或业务峰值明显的在线系统；
2. 对安全、可靠性或合规要求高的系统；
3. 有遗留改造、云迁移、微服务/事件化演进的系统。

每个项目准备一页事实表：业务背景、规模、角色、技术栈、核心约束、三个关键决策、替代方案、验证数据、故障与改进。没有真实数据时写可核验的相对变化或测试口径，不编造精确数字。
