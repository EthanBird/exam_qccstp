# 考前与考试日清单

## 报名后立即

- [ ] 报名状态显示缴费成功
- [ ] 保存报名信息和支付凭证
- [ ] 将 10 月 20 日设置为准考证打印提醒
- [ ] 确认日常练习使用官方机考支持的输入法

## 考前两周

- [ ] 完成至少两次“综合知识 + 案例分析”240 分钟连考
- [ ] 完成至少三篇 120 分钟论文，其中一篇在下午同一时段训练
- [ ] 综合知识形成稳定的时间分配，不在单题长期停留
- [ ] 案例答案按题号分点，能引用场景约束
- [ ] 三个论文主项目的事实卡和主题映射完成
- [ ] 身份证或社会保障卡在有效期内

## 准考证打印（10 月 20～23 日）

- [ ] 下载 PDF 并保存到手机/云端
- [ ] 打印至少两份，分开放置
- [ ] 核对姓名、证件号、资格名称
- [ ] 核对具体日期、批次和入场时间
- [ ] 核对考点地址，不只看学校名称
- [ ] 规划交通、停车和备用路线
- [ ] 阅读准考证上的允许用品和禁止事项
- [ ] 如需异地住宿，提前预订

## 考前一天

- [ ] 身份证件原件
- [ ] 纸质准考证
- [ ] 准考证允许携带的考试用品
- [ ] 简单饮水和午餐计划
- [ ] 关闭高强度刷题，只看错题、高频表和论文提纲
- [ ] 保证睡眠，不临时更换输入法和答题策略

## 综合知识 + 案例分析连考

- 先读机考页面规则，确认科目切换和交卷逻辑。
- 综合知识第一轮回答确定项，标记需要计算或复核的题。
- 不为一道题消耗过多时间；给案例分析保留计划内时长。
- 案例先通读问题再读材料，按题号分点。
- 方案题写“原因 + 措施 + 机制 + 指标”，避免只列名词。
- 交卷前检查是否存在空题、漏题、单位错误和题号错位。

## 论文

- 选最贴合真实项目且能覆盖全部小问的题，不选“看起来最新”的题。
- 先列提纲，再写摘要和正文；保留检查时间。
- 全文保持项目、时间、角色、技术和数据一致。
- 每个核心决策至少包含场景依据、工作机制和验证结果。
- 不复制背诵模板，不写无法自圆其说的精确数字。
- 以考试页面的字数、保存和提交提示为准。

## 异常处理

遇到考试机、输入法或网络异常：

1. 立即举手报告监考人员；
2. 不自行重启、换机或反复操作；
3. 记住异常发生的大致时间和当前题号；
4. 按监考人员安排处理并确认作答是否保存。
